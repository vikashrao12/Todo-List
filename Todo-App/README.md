# To-Do List Application

This is a simple To-Do List application built using **React** and **Vite** as part of an assignment.

The application allows users to:
- Add new tasks
- Edit existing tasks
- Mark tasks as completed
- Delete tasks

---

## Technologies Used

- React (Functional Components)
- Vite
- JavaScript
- CSS

---

##  Project Features

- Uses functional components
- State is managed in the App component
- Props are used to pass data between components
- Tasks are rendered dynamically using the map function
- Each task has a unique key
- Clean and simple user interface

---

##  How to Run the Project
- npm install
- npm run dev
